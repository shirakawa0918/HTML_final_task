# HTML Final Task（Level1）

## サイト概要
本サイトは、既存のWebサイトテンプレートをベースにしつつ、
Bootstrapを活用してレイアウトを再設計し、
オリジナルのWebサイトとして制作したものです。

元のアニメ系サイト構成を参考にしながら、
コンテンツを「League of Legends」の
非公式紹介サイトへ書き換えています。

レスポンシブデザインを維持しつつ、
配置や見え方を中心に改善を行いました。

---

## 使用技術
- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript（一部）

---

## レイアウト・構成の変更点

### 共通レイアウト
- ヘッダー（ナビゲーションバー）を**fixed**に変更し、スクロールしても常に画面上部に表示されるようにした
- 固定ヘッダーに対応するため、各ページのコンテンツがヘッダーに隠れないよう**上部に十分な余白**（padding-top）を設けた
- 全ページで共通のナビゲーションバーを使用し、サイト全体のレイアウトを統一
- ヘッダー以外の各コンテンツ要素からもページ遷移できるようリンクを配置し、導線を整理・改善

### index.html（トップページ）
- トップページを**2カラムレイアウト**に変更し、情報を整理して視認性向上
- コンテンツを意味のある**セクションごとに分割**し、構造が一目で分かりやすい配置に調整

### character.html（チャンピオン一覧）
- チャンピオン一覧を**グリッドレイアウト**で表示し、複数のカードを整然と並べ見やすくした
- 一覧ページの全体構成を「**左：チャンピオン一覧** ／ **右：サイドエリア**」の2カラムレイアウトに変更
- 検索・チェック用のUI（※現在はダミー）を右側のサイドエリアにまとめて配置し、レイアウトを整理

### character-yone.html（チャンピオン詳細ページ）
- 一覧ページとは明確に異なる**詳細ページ専用のレイアウト**を採用
- チャンピオン画像と説明文を**2カラム構成**で横並びに配置し、情報をすっきり整理
- 見出しや説明文を**ブロック単位**でグループ化して視覚的なまとまりを強化

## 実装面での改善点

- 各HTMLファイル内に記述されていた `style` タグを削除
- CSSを `base.css` に集約し、可読性・保守性を向上
- インデントを統一し、コードの可読性を改善
- 使用されていないCSSや不要なコードを削除
