# HTML Final Task

## Webサイト概要

本Webサイトは、配布されたテンプレートサイトを元に、  
**既存のページ構成・大まかなレイアウトを崩さず**、  
レイアウトや配置を工夫してオリジナルサイトとして制作したものです。

ページの削除は行わず、Bootstrap を活用して  
PC・タブレット・スマートフォンに対応した  
レスポンシブレイアウトを維持しています。

---

## 変更したレイアウト・構造（6点）

※色の変更ではなく、**配置や見え方の変更**を中心に行っています。

### 1. ナビゲーションバーのレイアウト変更
- ナビゲーションバーの高さと余白を調整し、画面上部での視認性を向上
- ロゴとメニューの配置を整理し、情報が把握しやすい構造に変更

---

### 2. トップページのメインセクション構成変更
- テキストのみの構成から、見出しを強調したヒーローセクション風の配置に変更
- コンテンツを中央寄せにし、ページを開いた際に内容が伝わりやすい構成にした

---

### 3. カードコンテンツの配置変更
- Bootstrap のグリッドシステムを使用し、カードを横並びで表示
- 画面サイズに応じて自動的に縦並びへ切り替わるよう調整し、レスポンシブ性を向上

---

### 4. Signup / Login フォームのレイアウト変更
- フォーム全体を中央に配置し、入力に集中しやすいレイアウトに変更
- 入力欄とボタンの間隔を調整し、操作性を向上させた

---

### 5. フッター構造の変更
- フッター内の情報を整理し、ページ下部として分かりやすい構造に変更
- コピーライト表記を中央に配置し、視認性を向上

---

### 6. レスポンシブデザインの調整
- スマートフォン表示時に余白が狭くなりすぎないよう調整
- Bootstrap のクラスを活用し、画面サイズごとに適切な余白と配置を設定

---

## その他の対応内容

- Webサイトの内容に合うファビコンへ変更（※レイアウト変更には含めていない）
- インデントを揃え、意味のあるクラス名を付けることでコードの可読性を向上
- 使用されていないCSSや不要なコードを削除
- クローン元で確認できた細かなレイアウト崩れを修正
